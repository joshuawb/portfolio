Complex Survey Analysis: Vitamin D vs. Bone Mineral Density

This project was completed as part of a course at University. Analysing data from complex surveys using the survey package in R for data from NHANES. This analysis investigates, using weighted estimation of totals as well as weighted regression, the relationship between vitamin D deficiency and spine bone mineral density. Literature review, code, and results are in the same repository.

Rice Grain Classification

This project was also completed as part of a course at University. Identification of rice grains, using Multivariate Data Analysis techniques applied in R, based on 106 features extracted from 75,000 pictures. Methods used include LDA, QDA, Random Forest, and XGBoost. 99.96% accuracy achieved on 30% validation set using Random Forest.

  • view project on github

Walmart Sales Forecasting

This project involved predicting sales for Walmart stores using time series analysis and machine learning techniques. The dataset included historical sales data, promotions, and economic indicators. Various models were tested, including ARIMA and XGBoost.

Current Role

As a Data Scientist at timbre Digital, I design and deliver high-impact, end-to-end data solutions that shape product direction and drive measurable business outcomes. I work across the full data science lifecycle. This can range from wrangling messy, high-volume datasets all the way through to deploying production-ready machine learning models. I bring technical depth, clear communication, and strong commercial intuition to every project.

What I do:

  • Transform complex, multi-source datasets into clean, reliable analytical assets that reveal patterns, behaviours, and opportunities.
  • Develop, train, and tune machine learning models for prediction, classification, optimisation, and customer behaviour insights.
  • Maintain and continuously improve production models, ensuring they remain accurate, robust, and aligned with evolving business needs.
  • Design and refine automated data pipelines that support scalable, repeatable analysis and model operations.
  • Partner with cross-functional teams to translate business problems into data-driven solutions with clear technical approaches.
  • Communicate insights and model outcomes to stakeholders with clarity whether through presentations, dashboards, or written reports.
  • Document models, pipelines, and processes to support transparency, reproducibility, and long-term maintainability.
  • Continuously adopt new tools, methods, and modelling techniques to push the quality and efficiency of our workflows.
  • Support deployment workflows, including refactoring scripts, managing batch jobs, and operationalising model outputs.
  • Independently manage priorities, project timelines, and day-to-day execution in a fast-paced analytics environment.
  • Build dashboards, reports, and visual analytics in Tableau to surface insights and support strategic decision-making.
