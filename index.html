Joshua Buchanan - Data Scientist

Joshua Buchanan is currently a Data Scientist at timbre Digital and Statistics Master's Student at The University of Auckland. This webpage is designed to showcase personal projects and key learnings from studies and acts as a portfolio page for potential employers to view to get an idea of his skills and interests.

Step Count vs. Weather

Investigation on the relationship between step count, as recorded by my Apple Watch over the last two years, and Auckland temperature and precipitation in Python.

  • view project on github

Rice Grain Classification

Identification of rice grains, using Multivariate Data Analysis techniques applied in R, based on 106 features extracted from 75,000 pictures. Methods used include LDA, QDA, Random Forest, and XGBoost. 99.96% accuracy achieved on 30% validation set using Random Forest.

  • view project on github

Covid SQL Data Exploration

A showcase of SQL skills using Covid-19 datset. Key skills used: Joins, CTE's, Temp Tables, Windows Functions, Aggregate Functions, Creating Views, and Converting Data Types.

  • view project on github

Complex Survey Analysis: Vitamin D vs. Bone Mineral Density

Analysing data from complex surveys using the survey package in R for data from NHANES. This analysis investigates, using weighted estimation of totals as well as weighted regression, the relationship between vitamin D deficiency and spine bone mineral density. Literature review, code, and results are in the same repository.
