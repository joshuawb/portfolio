Course Outline – STATS 762 : Regression for Data Science
2024 Semester One (15 points)

Grade Achieved

A-

Course Prescription

Application of the generalised linear model to fit data arising from a wide range of sources, including multiple linear regression models, Poisson regression, and logistic regression models. Graphical data exploration. Model building for prediction and causal inference. Other regression models such as quantile regression.

Course Overview

This course extends on regression techniques from STATS 201/208 with emphasis on practical application. Key topics include estimation, diagnostics, inference, simulation-based methods (bootstrapping, cross-validation), and regression for prediction and explanation. Causal inference, model selection, penalised regression (lasso), and ethics in predictive modelling are also introduced. Students gain tools applicable across academia and industry, including data science, economics, and social sciences.

Learning Outcomes

By the end of the course, students will be able to:

  1. Describe the components of the GLM and select appropriate models for different data types (Capabilities 3, 4, 5, 8).
  2. Apply diagnostic procedures and propose solutions for model issues (Capabilities 3, 4, 5).
  3. Use cross-validation to select predictive models and assess model performance (Capabilities 3, 4, 5).
  4. Create and interpret causal graphs, identifying confounding variables (Capabilities 3, 4, 5).
  5. Use bootstrapping to evaluate sampling distributions in GLMs (Capabilities 3, 4, 5).
  6. Answer questions about relationships between predictors and response using fitted models (Capabilities 3, 4, 5).
  7. Explore and visualise variable relationships using appropriate graphical techniques (Capabilities 3, 4, 5).
  8. Explain ethical implications of predictive modelling, including misuse or bias (Capabilities 3, 4, 5, 8).
