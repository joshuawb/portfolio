Course Prescription

Professional skills, advanced statistical programming, numerical computation and graphics.

Course Overview

This course teaches advanced statistical programming in R, numerical computation and graphics—skills vital for statistical research and for developing R extension packages. Students gain practical expertise applicable to data science and statistical computing careers. Topics span R basics, control flow, matrices, numerical methods, graphics, object‑oriented systems (S3/S4), package development, efficiency considerations, and scoping.

Learning Outcomes

By the end of the course, students will be able to:

  1. Apply core R concepts such as vectorisation, recycling, and scoping (Capabilities 3, 5).
  2. Write concise R functions for specialised tasks (Capabilities 3, 4, 5).
  3. Develop efficient, readable code for data‑analytic workflows (Capabilities 3, 4, 5).
  4. Import, process, and export data in human‑readable formats (Capabilities 3, 4, 5, 8).
  5. Create informative graphics using low‑ and high‑level R functions (Capabilities 3, 5, 6, 8).
  6. Design object‑oriented systems with S3 and/or S4 (Capabilities 4, 5, 6, 7, 8).
  7. Build and submit an R package to CRAN (Capabilities 1, 3, 4, 5, 6, 7, 8).
